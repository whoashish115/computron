Hi! I'm **[whoashish115](https://whoashish.com)**, and this is my personal **Computer Science** wiki.

I built **Computron** to keep everything I learn about core computer science in one place - from programming fundamentals, data structures and algorithms, operating systems, database management systems, computer networks, software engineering, computer organization and architecture, compiler design, theory of computation, distributed systems, cloud computing, computer security, parallel and concurrent computing, and everything in between. Along the way, I also collect lecture notes, implementation details, code snippets, interview insights, references, books, papers, resources, and small concepts that I don't want to forget. Instead of searching the internet for the same topics repeatedly, I can simply come back here.

If you're visiting for the first time, I'd recommend starting with the sidebar navigation. Everything is organized by subject, making it easy to explore programming, data structures, algorithms, discrete mathematics, operating systems, databases, networking, software engineering, computer architecture, digital logic, compiler design, theory of computation, distributed systems, cloud computing, cybersecurity, embedded systems, programming languages, computer graphics, human-computer interaction, and more.

Most of these notes are written for my future self, but if they help you understand a concept, prepare for interviews, strengthen your computer science fundamentals, or learn something new, that's even better. **Computron** is a living knowledge base that will continue to grow as I study, build, and explore the foundations of computer science.
